# Unity 버전 검증

## 상태

- 보류(2022.3.62f3 미설치)

## 복원 수준 목표

- R1

## 목표

- 새 `UnityProject/`가 2022.3.62f3에서 안전하게 열리는지 확인한다.
- 문제가 발생하면 2022.3.2f1 환경과 비교한다.

## 현재 근거

- 가이드 우선 버전: 2022.3.62f3
- 가이드 원본 확인 버전: 2022.3.2f1
- ExportedProject 기록 버전: 2022.3.62f3
- 로컬 설치: 2022.3.2f1, 6000.3.17f1

## 완료 조건

- [ ] 2022.3.62f3 설치
- [ ] 새 프로젝트 에디터 로드
- [ ] 패키지/Console 상태 기록
- [ ] 빈 Scene 저장/재로드
- [ ] 버전 선택 근거 갱신
- [ ] 사용자 승인 시에만 Windows 빌드 검증

## 위험과 되돌리기

- 위험: 첫 실행에서 ProjectSettings와 manifest가 자동 변경될 수 있음
- 되돌리기: 실행 전 새 프로젝트 세 폴더를 별도 보존하고 diff 비교
